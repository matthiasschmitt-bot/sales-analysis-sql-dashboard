# 📊 Sales Analysis SQL Dashboard

Willkommen auf der Projektseite des **Sales Analysis SQL Dashboard**.  
Dieses Projekt demonstriert einen vollständigen Mini-Analytics-Workflow mit SQL, CSV-Daten und Dashboard-Konzept.

---

## 🔍 Projektüberblick

- Aufbau eines relationalen Datenmodells (Produkte, Kunden, Verkäufe)
- Import von CSV-Daten in PostgreSQL
- Erstellung von SQL-Abfragen für Analysen
- Konzeption eines Dashboards für Management-Entscheidungen

---

## 📈 Wichtige Analysen

- Gesamtumsatz
- Top-Produkte
- Umsatz nach Region
- Umsatz pro Monat

Details findest du im Ordner `analysis/`.

---

## 📊 Dashboard-Konzept

Das geplante Dashboard umfasst:

- KPI-Karten
- Diagramme
- Tabellen

Mehr dazu im Ordner `dashboard/`.

---

## 🛠️ Technologien

- PostgreSQL
- SQL
- CSV-Daten
- (optional) Power BI oder Excel
